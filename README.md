## About

A Jekyll theme with sidebar for personal website and blog.

![Mockup](/assets/img/mockup.png)

## Features

- Responsive
- Semantic HTML
- Syntax Highlight
- Search
- Reading time
- Mermaid Diagrams
- Social sharing link
- Custom 404 page
- Next and Previous post
- Back to top button
- Table of contents
- Feed RSS
- Sitemap
- SEO Optimization
